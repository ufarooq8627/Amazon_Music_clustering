# Amazon Music Clustering

A small project that performs clustering analysis on Amazon music data and provides an interactive Streamlit UI.

## Project files
- `music_clustering.ipynb` - Main analysis notebook that:
  - Loads and inspects the raw dataset (`single_genre_artists.csv`)
  - Cleans and preprocesses audio/artist features
  - Scales features, applies PCA for 2D visualization
  - Runs K-Means clustering and profiles clusters
  - Exports `Final_Amazon_Music_Project.csv` (used by the app)
- `streamlit_app.py` - Streamlit application that:
  - Loads the processed CSV via a cached `load_data` helper
  - Provides sidebar controls (filters, "Show Raw Data")
  - Displays a PCA 2D cluster scatter and cluster profile visualizations (Plotly/Matplotlib)
- `app_v2.py` - Alternate Streamlit layout / visualization (optional)
- `Final_Amazon_Music_Project.csv` - Final processed dataset generated by the notebook
- `single_genre_artists.csv` - Original dataset with artist information

## Requirements

Install the recommended packages:
```bash
pip install pandas numpy matplotlib seaborn scikit-learn streamlit plotly jupyterlab
```

## Quick start

1. Confirm `Final_Amazon_Music_Project.csv` is present in the project root. If missing, re-run `music_clustering.ipynb` to regenerate it.
2. Reproduce the analysis:
   - Open `music_clustering.ipynb` in Jupyter and run the cells.
3. Run the Streamlit app:
```bash
streamlit run streamlit_app.py
```

## Notes & Usage

- The Streamlit app expects `Final_Amazon_Music_Project.csv` to include PCA columns (`PC1`, `PC2`) and cluster labels as produced by the notebook. If you recompute PCA or re-label clusters, keep the column names consistent or update the app accordingly.
- If the app cannot find the CSV it will raise a FileNotFoundError; ensure the file exists in the repository root.

## Troubleshooting

- FileNotFoundError: Verify `Final_Amazon_Music_Project.csv` exists and has correct permissions.
- Missing packages: Install dependencies shown above.
- If you modify the notebook and regenerate data, restart Streamlit to pick up changes.
